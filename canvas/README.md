# Canvas 学习项目

这是一个可直接打开的静态学习项目，包含以下内容：

- `index.html`：总入口
- `docs/`：独立文档菜单与完整概念讲解
- `native/`：原生 JavaScript 学习站
- `vue/`：Vue 3 CDN 学习站
- `react/`：React 18 CDN 学习站
- `examples/`：三种技术栈对应的示例代码文件

## 学习章节

1. 基础绘制
2. 样式与文本
3. 变换与状态栈
4. 图像与像素
5. 动画与交互
6. 自我练习模块

## 打开方式

直接双击 `index.html` 即可开始学习。

如果浏览器对本地脚本有额外限制，也可以在 `canvas/` 目录启动一个简单静态服务：

```bash
python3 -m http.server 4173
```

然后访问 `http://localhost:4173`。
