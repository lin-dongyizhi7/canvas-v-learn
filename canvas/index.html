Canvas 学习项目

一套围绕 Canvas 常见 API、概念讲解、代码示例和练习模块搭建的学习站。 你可以从 Vue、React、原生 JS 三种视角进入同一套知识地图。

你会在这里学到什么

常见 API 全景

从基础图形、样式文本、变换、像素处理到动画交互，章节成体系排列。

三套前端实现

同一份知识，用 Vue、React、原生 JS 分别讲一遍，不让框架差异偷走理解。

练习模块

提供可直接动手的自由画布，适合你继续补颜色、橡皮擦、导出等功能。

学习建议

  1. 先看独立文档菜单，了解章节地图和重点 API。
  2. 再任选一种技术栈进入学习站，边看概念边看预览。
  3. 最后回到练习模块，自己把功能往上堆，记忆会比死背牢很多。
